This is just a nice UI to show a key and label

## Client exports
**Show Text UI**
```lua copy
---@param label string
---@param key string
---@param position string
exports.mt_lib:showTextUI(label, key, position)
```
- label: `string`
    - UI Label
- key: `string`
    - UI Key
- position: `string`
    - UI Position on screen: `left` | `right` | `bottom` | `top`

**Hide Text UI**
```lua copy
exports.mt_lib:hideTextUI()
```
