# MT Nitrous
Vehicle nitrous/NOS system for QBCore, QBox and ESX

**Features:**
- Nitrous bottle item to install at vehicle (option to only be able to install with turbo mod installed at vehicle)
- Save nitrous level and purge color at SQL
- Boost and purge modes
- Option to change boost level (can add as many levels as you want at config)
- Option to change purge smoke color
- Simple nitrous level indicator UI included (design can be changed at css file)
- Sound effects included
- Full optimized (0.3ms in boost use, when not in use 0.0ms, when not in use inside of vehicle 0.0ms)
