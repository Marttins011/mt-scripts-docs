# Create blip
Just create a blip with sent info

## Client exports

Create blip
```lua copy
exports.mt_lib:createBlip(coords, sprite, display, scale, color, label)
```
- coords: `table` (`vector`)
    - Coordinates to create the blip
- sprite: `number` (`integer`)
    - Blip sprite
- display: `number` (`integer`)
    - Blip display
- scale: `number` (`float`)
    - Blip scale
- color: `number` (`integer`)
    - Blip color
- label: `string`
    - Blip label
- return: `number`
    - Blip index
