## Client exports
**Show NPC Dialogue**
```lua copy
exports.mt_lib:showDialogue(data)
```
- data: `table` (`object`)
    - ped: `number` (`integer`)
        - Target ped entity
    - label: `string`
        - Dialogue Title
    - speech: `string`
        - Dialogue description
    - options: `table` (`object`)
        - id: `string` | `number`
            - Option index
        - label: `string`
            - Option label
        - icon: `string`
            - Option Icon from Fontawesome
        - close: `boolean`
            - If the UI should close after using the option
        - action: `function`
            - Function to execute when clicking the option
