# MT Containers
MT Containers is a simple containers/storage units creation script for QBCore and QBXCore

**Features**
- Create new containers with a menu ingame
- Edit created containers with a menu ingame
- Delete created containers with a menu ingame
- Buy containers for a especified price
- Use code to access containers
- Change code whenever you want
- Sell containers by a percentage of its value
- Transfer containers to other players
- See containers available for sell at GPS using a command
- Full logs system trought discord webhook

**Dependicies**
- ox_inventory or qb-inventory
- ox_target or qb-target
- ox_lib
- oxmysql
- qbx_core or qb-core
